Zweck der Website

Checkuin ist eine nicht-kommerzielle Plattform für Bildungszwecke. Die Anwendung dient der Demonstration KI-gestützter Symptomanalyse und ersetzt keinesfalls eine ärztliche Untersuchung oder medizinische Beratung.

Haftungsausschluss

Die auf dieser Website bereitgestellten Informationen dienen ausschließlich allgemeinen Informationszwecken. Sie stellen keine medizinische Beratung, Diagnose oder Behandlungsempfehlung dar. Bei gesundheitlichen Beschwerden wenden Sie sich bitte an eine Ärztin oder einen Arzt.

Technologie & Hosting

Frontend: Angular 21 · Backend: Spring Boot 3 / Java 17
Datenbank: PostgreSQL · KI: Bayesianische Inferenz
Hosting: Lokale Entwicklungsumgebung / Docker
